# AI-ассистент методиста

Веб-инструмент, который собирает методические материалы по физике для российской школы — конспекты уроков, проверочные и лабораторные работы — в логике **ФГОС**. Работает на **Claude** (Anthropic).

![Скриншот](docs/screenshot.png)

## Зачем

Я несколько лет работал методистом и написал 19 методических пособий полного цикла — по физике, биологии и химии. Составление материала под ФГОС — это каждый раз одна и та же рутина: планируемые результаты трёх видов, цель и задачи, этапы урока, оборудование. Проект автоматизирует именно её: методист задаёт тему, класс и тип материала, а модель собирает готовый черновик в правильной структуре — остаётся вычитать и адаптировать под свой класс.

## Как это работает

Вся экспертиза вынесена в промпты (`app/prompts.py`): системный промпт задаёт роль методиста и требования ФГОС — обязательные предметные, метапредметные и личностные результаты, системно-деятельностный подход, — а по типу материала подставляется своя инструкция со структурой. Запрос уходит в Claude через официальный SDK, ответ печатается в браузере по мере генерации.

Провайдер спрятан за одной функцией в `app/llm.py` — сменить модель или провайдера можно в одном месте.

```
Форма (тема, класс, тип)
        │
        ▼
  FastAPI  ── build_user_prompt() ──►  Claude (anthropic SDK, стриминг)
        │                                    │
        ◄──────────── поток текста ──────────┘
        ▼
  Материал печатается в браузере вживую
```

## Что умеет

- Три типа материала: конспект урока, проверочная работа, лабораторная работа.
- Классы 7–11, тема — любая; примеры тем подставляются в один клик.
- Планируемые результаты по ФГОС включаются автоматически.
- Потоковая генерация: материал печатается на экране по мере готовности, до первого токена показывается скелет-заглушка.
- Готовый материал можно скопировать, скачать в Markdown или отправить на печать и в PDF.
- **Демо-режим без ключа**: приложение запускается и показывает пример-заглушку, так что его можно посмотреть без API-ключа и без затрат. Режим виден по бейджу в шапке.

## Стек

- **Python 3.12**, FastAPI и uvicorn.
- Официальный SDK **`anthropic`**, модель `claude-opus-4-8`, потоковый ответ.
- Ванильный JavaScript и [marked](https://marked.js.org/) для рендера Markdown, без сборки.

## Запуск

```bash
pip install -r requirements.txt
```

Для реальной генерации создайте `.env` из примера и впишите ключ:

```bash
cp .env.example .env      # затем ANTHROPIC_API_KEY
```

Запуск (без ключа откроется демо-режим):

```bash
python -m app.main
```

Открыть **http://localhost:8000**.

## Структура

```
ai-methodist/
├── app/
│   ├── prompts.py   # системный промпт (ФГОС) + сборка запроса по типу материала
│   ├── llm.py       # провайдер: Claude через SDK или демо-заглушка без ключа
│   └── main.py      # FastAPI: форма, потоковая и обычная генерация, статус
├── web/
│   └── index.html   # интерфейс: форма + рендер Markdown
├── docs/
│   └── screenshot.png
├── .env.example
└── requirements.txt
```
